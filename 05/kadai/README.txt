仕様
1個目の課題はkadai01ディレクトリにあります。
2個目の発展課題はkadai02ディレクトリにあります。
Makefileはkadaiディレクトリにおいてあります。
1個目の課題は $ make optimizer1
2個目の課題は $ make optimizer2
でビルドできます。


課題1
optimizer1の実行結果によると推測される富士山頂の7月の平均気温は摂氏3.971240723度である。

課題2
重回帰分析により、家賃を目的変数とし、「徒歩（分）,専有面積（㎡）,築年数（年）,階数（階）,新築,二階以上,南向き,駐車場,オートロック,エアコン,バストイレ別,追い焚き,フローリング,ペット相談可」を説明変数とした。
このデータは./kadai02/houserent_data.csvにある。このファイルを読み込み分析を行う。Momentum有りのミニバッチ勾配法により最適パラメータを求める。
初めてソースファイルを分割ということでexternを使ってみたかったので、各関数にデータ配列を渡すのではなく、グローバル変数で定義するという仕様にあえてした。
（しかし、ヘッダファイルで大域変数を宣言するやり方など色々あるらしく、ネットで調べてもどれが正しい情報かはっきりしなかったので、
授業でそのあたりを詳しく取り扱っていただけるとうれしいです。）
結果：
houserent =   -1113.7409 * x_1 + 2707.5555 * x_2 + -574.8958 * x_3 + 1445.7099 * x_4 + 4727.1572 * x_5 + -1615.4776 * x_6
